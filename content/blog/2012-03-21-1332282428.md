<p><a href="https://www.meganii.com/blog/2012/03/20/1332214416">TEDの英語原稿を取得する - ギークを夢見るじょーぶん男子</a>で取得した英語原稿をダウンロードしたmp3ファイルに埋め込みたい!!</p><br />

<div class="section">
    <h4>id3lib-<a class="keyword" href="http://d.hatena.ne.jp/keyword/ruby">ruby</a>を利用してmp3に歌詞情報を埋め込む</h4>
    <p><a href="http://id3lib-ruby.rubyforge.org/" target="_blank"><img class="alignleft" align="left" border="0" src="http://capture.heartrails.com/150x130/shadow?http://id3lib-ruby.rubyforge.org/" alt="" width="150" height="130" /></a><a style="color:#0070C5;" href="http://id3lib-ruby.rubyforge.org/" target="_blank">id3lib-ruby - ID3 tag library for Ruby</a><a href="http://b.hatena.ne.jp/entry/http://id3lib-ruby.rubyforge.org/" target="_blank"><img border="0" src="http://b.hatena.ne.jp/entry/image/http://id3lib-ruby.rubyforge.org/" alt="" /></a><br><strong>mp3のタグを扱うライブラリid3libのラッパー</strong><br style="clear:both;" /><br></p><br />
<p></p>

<div class="section">
    <h5>準備</h5>
    <pre class="code" data-lang="" data-unlink>brew install id3lib
gem install id3lib-ruby</pre>
</div>
<div class="section">
    <h5>Lyricsに英語原稿を設定</h5>
    <pre class="code lang-ruby" data-lang="ruby" data-unlink><span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">rubygems</span><span class="synSpecial">'</span>
<span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">id3lib</span><span class="synSpecial">'</span>

<span class="synComment"># TEDからダウンロードしたmp3ファイルの読み込み</span>
tag = <span class="synType">ID3Lib</span>::<span class="synType">Tag</span>.new(<span class="synSpecial">'</span><span class="synConstant">ted.mp3</span><span class="synSpecial">'</span>)
<span class="synComment"># 英語原稿を歌詞情報として登録</span>
tag.lyrics = <span class="synSpecial">&quot;</span><span class="synConstant">TEDから取得した英語原稿</span><span class="synSpecial">&quot;</span>
<span class="synComment"># 更新</span>
tag.update!
</pre>
</div>
</div>
