<p><a class="keyword" href="http://d.hatena.ne.jp/keyword/Python">Python</a>でのループで書き方ってどうするんだろうって思って、ちょっと調べてみました。</p>

<div class="section">
    <h4>通常のループ</h4>
    <pre class="code lang-python" data-lang="python" data-unlink><span class="synStatement">for</span> x <span class="synStatement">in</span> <span class="synIdentifier">range</span>(<span class="synConstant">0</span>,<span class="synConstant">10</span>):
    <span class="synIdentifier">print</span> x
</pre>
</div>
<div class="section">
    <h4>辞書のループ</h4>
    <pre class="code lang-python" data-lang="python" data-unlink>keys = [<span class="synConstant">u&quot;名前&quot;</span>, <span class="synConstant">u&quot;ヨミガナ&quot;</span>, <span class="synConstant">u&quot;Emailアドレス&quot;</span>, <span class="synConstant">u&quot;住所&quot;</span>]
values = [<span class="synConstant">u&quot;meganii&quot;</span>, <span class="synConstant">u&quot;メガニイ&quot;</span>, <span class="synConstant">u&quot;email@address&quot;</span>, <span class="synConstant">u&quot;住所&quot;</span>]

address = <span class="synIdentifier">dict</span>()
<span class="synStatement">for</span> key, value <span class="synStatement">in</span> <span class="synIdentifier">zip</span>(keys, values):
    address[key] = value


<span class="synStatement">for</span> key, value <span class="synStatement">in</span> address.iteritems():
    <span class="synIdentifier">print</span> key, value
</pre><p>dict(...)を使うと、キーを''で囲わなくても良いみたい<br />
<br />
</p>

</div>
<div class="section">
    <h4>ループ内でインデックス値を使いたい場合</h4>
    <pre class="code lang-python" data-lang="python" data-unlink><span class="synStatement">for</span> i, x <span class="synStatement">in</span> <span class="synIdentifier">enumerate</span>(<span class="synIdentifier">range</span>(<span class="synConstant">0</span>,<span class="synConstant">10</span>)):
  <span class="synIdentifier">print</span> i
  <span class="synIdentifier">print</span> x
</pre>
</div>
<div class="section">
    <h4>zipとenumerateを同時に使う</h4>
    <pre class="code lang-python" data-lang="python" data-unlink><span class="synStatement">for</span> i, (key,value) <span class="synStatement">in</span> <span class="synIdentifier">enumerate</span>(<span class="synIdentifier">zip</span>(keys, values)):
     <span class="synIdentifier">print</span> i
     <span class="synIdentifier">print</span> key
     <span class="synIdentifier">print</span> value
</pre>
</div>
