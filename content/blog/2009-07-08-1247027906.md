    <h4>自分へのメモ</h4>
    <p>データせっせと吐き出したはいいけれど、それをグラフで可視化したい場合の処理をさせたかった。以下のページを参考に書いてみた。</p>
<pre class="code" data-lang="" data-unlink>#!/bin/sh

# `seq 0 N`で0-29までの整数を読み込み

for i in `seq 0 29`
do
#---シェル内でのgnuplot起動
#### EOF行までの間に空行を挟んではいけない。
gnuplot &lt;&lt;EOF
set out &#34;result$i.png
set terminal png
plot &#39;result$i.dat&#39; u 1:3 w l ti &#39;bestValue&#39;, &#39;result$i.dat&#39; u 1:4 w l ti &#39;fitnessSol&#39;,&#39;result$i.dat&#39; u 1:6 w l ti &#39;minBar&#39;,&#39;result$i.dat&#39; u 1:8 w l ti &#39;fitnessCri&#39;
EOF
done</pre>
