<div class="section">
    <h4>サンプルを修正</h4>
    <p>以下のリンクから<a class="keyword" href="http://d.hatena.ne.jp/keyword/Ruby">Ruby</a>のサンプルを取得。oauthフォルダに入ってるものが、Web Applicationのサンプルみたいです。Web Applicationの<a class="keyword" href="http://d.hatena.ne.jp/keyword/Ruby">Ruby</a>版のサンプルは、<a class="keyword" href="http://d.hatena.ne.jp/keyword/sinatra">sinatra</a>を使ったものでした。<br />
<a href="http://evernote.s3.amazonaws.com/api/evernote-api-1.20.zip">http://evernote.s3.amazonaws.com/api/evernote-api-1.20.zip</a></p><br />
<br />
<p>自分の取得した<a class="keyword" href="http://d.hatena.ne.jp/keyword/API">API</a>キーを反映するために、<a class="keyword" href="http://d.hatena.ne.jp/keyword/evernote">evernote</a>_config.rbを 修正</p>
<pre class="code lang-ruby" data-lang="ruby" data-unlink><span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">oauth</span><span class="synSpecial">'</span>
<span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">oauth/consumer</span><span class="synSpecial">'</span>
<span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">thrift</span><span class="synSpecial">'</span>

<span class="synComment"># 以下、取得したAPIキー</span>
<span class="synType">OAUTH_CONSUMER_KEY</span> = <span class="synSpecial">&quot;</span><span class="synConstant">YOUR_CONSUMER_KEY</span><span class="synSpecial">&quot;</span>
<span class="synType">OAUTH_CONSUMER_SECRET</span> = <span class="synSpecial">&quot;</span><span class="synConstant">YOUR_CONSUMER_SECRET</span><span class="synSpecial">&quot;</span>

<span class="synComment"># 定数</span>
<span class="synType">EVERNOTE_SERVER</span> = <span class="synSpecial">&quot;</span><span class="synConstant">https://sandbox.evernote.com</span><span class="synSpecial">&quot;</span>
<span class="synType">REQUEST_TOKEN_URL</span> = <span class="synSpecial">&quot;#{</span><span class="synType">EVERNOTE_SERVER</span><span class="synSpecial">}</span><span class="synConstant">/oauth</span><span class="synSpecial">&quot;</span>
<span class="synType">ACCESS_TOKEN_URL</span> = <span class="synSpecial">&quot;#{</span><span class="synType">EVERNOTE_SERVER</span><span class="synSpecial">}</span><span class="synConstant">/oauth</span><span class="synSpecial">&quot;</span>
<span class="synType">AUTHORIZATION_URL</span> = <span class="synSpecial">&quot;#{</span><span class="synType">EVERNOTE_SERVER</span><span class="synSpecial">}</span><span class="synConstant">/OAuth.action</span><span class="synSpecial">&quot;</span>
<span class="synType">NOTESTORE_URL_BASE</span> = <span class="synSpecial">&quot;#{</span><span class="synType">EVERNOTE_SERVER</span><span class="synSpecial">}</span><span class="synConstant">/edam/note/</span><span class="synSpecial">&quot;</span>
</pre><p><br />
Evetenoのgemを使うように、ちょこちょこ修正。</p>
<pre class="code lang-ruby" data-lang="ruby" data-unlink><span class="synPreProc">require</span> <span class="synSpecial">&quot;</span><span class="synConstant">./evernote_config.rb</span><span class="synSpecial">&quot;</span>
</pre><p>としないと、読み込まれなかったのはなーぜー？</p>
<pre class="code lang-ruby" data-lang="ruby" data-unlink><span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">rubygems</span><span class="synSpecial">'</span>
<span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">sinatra</span><span class="synSpecial">'</span>
<span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">evernote</span><span class="synSpecial">'</span>
enable <span class="synConstant">:sessions</span>

<span class="synComment"># Load our dependencies and configuration settings</span>
<span class="synPreProc">require</span> <span class="synSpecial">&quot;</span><span class="synConstant">./evernote_config.rb</span><span class="synSpecial">&quot;</span>

<span class="synComment">##</span>
<span class="synComment"># Verify that you have obtained an Evernote API key</span>
<span class="synComment">##</span>
before <span class="synStatement">do</span>
  <span class="synStatement">if</span> (<span class="synType">OAUTH_CONSUMER_KEY</span> == <span class="synSpecial">&quot;</span><span class="synConstant">en-edamtest</span><span class="synSpecial">&quot;</span>) 
    halt <span class="synSpecial">'</span><span class="synConstant">&lt;span style=&quot;color:red&quot;&gt;Before using this sample code you must edit evernote_config.rb and replace OAUTH_CONSUMER_KEY and OAUTH_CONSUMER_SECRET with the values that you received from Evernote. If you do not have an API key, you can request one from &lt;a href=&quot;http://www.evernote.com/about/developer/api/&quot;&gt;http://www.evernote.com/about/developer/api/&lt;/a&gt;.&lt;/span&gt;</span><span class="synSpecial">'</span>
  <span class="synStatement">end</span>
<span class="synStatement">end</span>

<span class="synComment">##</span>
<span class="synComment"># Index page</span>
<span class="synComment">##</span>
get <span class="synSpecial">'</span><span class="synConstant">/</span><span class="synSpecial">'</span> <span class="synStatement">do</span>
  erb <span class="synConstant">:authorize</span>
<span class="synStatement">end</span>

<span class="synComment">##</span>
<span class="synComment"># Reset the session</span>
<span class="synComment">##</span>
get <span class="synSpecial">'</span><span class="synConstant">/reset</span><span class="synSpecial">'</span> <span class="synStatement">do</span>
  session[<span class="synConstant">:request_token</span>] = <span class="synConstant">nil</span>
  erb <span class="synConstant">:authorize</span>
<span class="synStatement">end</span>

<span class="synComment">##</span>
<span class="synComment"># Get temporary credentials and redirect the user to Evernote for authoriation</span>
<span class="synComment">##</span>
get <span class="synSpecial">'</span><span class="synConstant">/authorize</span><span class="synSpecial">'</span> <span class="synStatement">do</span>
  callback_url = request.url.chomp(<span class="synSpecial">&quot;</span><span class="synConstant">authorize</span><span class="synSpecial">&quot;</span>).concat(<span class="synSpecial">&quot;</span><span class="synConstant">callback</span><span class="synSpecial">&quot;</span>)

  <span class="synStatement">begin</span>
    consumer = <span class="synType">OAuth</span>::<span class="synType">Consumer</span>.new(<span class="synType">OAUTH_CONSUMER_KEY</span>, <span class="synType">OAUTH_CONSUMER_SECRET</span>,{
        <span class="synConstant">:site</span> =&gt; <span class="synType">EVERNOTE_SERVER</span>,
        <span class="synConstant">:request_token_path</span> =&gt; <span class="synSpecial">&quot;</span><span class="synConstant">/oauth</span><span class="synSpecial">&quot;</span>,
        <span class="synConstant">:access_token_path</span> =&gt; <span class="synSpecial">&quot;</span><span class="synConstant">/oauth</span><span class="synSpecial">&quot;</span>,
        <span class="synConstant">:authorize_path</span> =&gt; <span class="synSpecial">&quot;</span><span class="synConstant">/OAuth.action</span><span class="synSpecial">&quot;</span>})
    session[<span class="synConstant">:request_token</span>] = consumer.get_request_token(<span class="synConstant">:oauth_callback</span> =&gt; callback_url)
    redirect session[<span class="synConstant">:request_token</span>].authorize_url

  <span class="synStatement">rescue</span> <span class="synType">Exception</span> =&gt; e
    <span class="synIdentifier">@last_error</span> = <span class="synSpecial">&quot;</span><span class="synConstant">Error obtaining temporary credentials: </span><span class="synSpecial">#{</span>e.message<span class="synSpecial">}&quot;</span>
    erb <span class="synConstant">:error</span>
  <span class="synStatement">end</span>
<span class="synStatement">end</span>

<span class="synComment">##</span>
<span class="synComment"># Receive callback from the Evernote authorization page and exchange the</span>
<span class="synComment"># temporary credentials for an token credentials.</span>
<span class="synComment">##</span>
get <span class="synSpecial">'</span><span class="synConstant">/callback</span><span class="synSpecial">'</span> <span class="synStatement">do</span>
  <span class="synStatement">if</span> (params[<span class="synSpecial">'</span><span class="synConstant">oauth_verifier</span><span class="synSpecial">'</span>].nil?)
    <span class="synIdentifier">@last_error</span> = <span class="synSpecial">&quot;</span><span class="synConstant">Content owner did not authorize the temporary credentials</span><span class="synSpecial">&quot;</span>
    erb <span class="synConstant">:error</span>
  <span class="synStatement">else</span>
    oauth_verifier = params[<span class="synSpecial">'</span><span class="synConstant">oauth_verifier</span><span class="synSpecial">'</span>]

    <span class="synStatement">begin</span>
      access_token = session[<span class="synConstant">:request_token</span>].get_access_token(<span class="synConstant">:oauth_verifier</span> =&gt; oauth_verifier)
      shard_id = access_token.params[<span class="synSpecial">'</span><span class="synConstant">edam_shard</span><span class="synSpecial">'</span>]

      <span class="synComment"># Construct the URL used to access the user's account</span>
      noteStoreUrl = <span class="synType">NOTESTORE_URL_BASE</span> + shard_id
      noteStoreTransport = <span class="synType">Thrift</span>::<span class="synType">HTTPClientTransport</span>.new(noteStoreUrl)
      noteStoreProtocol = <span class="synType">Thrift</span>::<span class="synType">BinaryProtocol</span>.new(noteStoreTransport)
      noteStore = <span class="synType">Evernote</span>::<span class="synType">EDAM</span>::<span class="synType">NoteStore</span>::<span class="synType">NoteStore</span>::<span class="synType">Client</span>.new(noteStoreProtocol)

      <span class="synComment"># Build an array of notebook names from the array of Notebook objects</span>
      notebooks = noteStore.listNotebooks(access_token.token)

      result = <span class="synType">Array</span>.new
      notebooks.each <span class="synStatement">do</span> |<span class="synIdentifier">notebook</span>| 
        result &lt;&lt; notebook
      <span class="synStatement">end</span>
      <span class="synIdentifier">@notebooks</span> = result
      erb <span class="synConstant">:complete</span>
    <span class="synStatement">rescue</span> <span class="synType">Exception</span> =&gt; e
      <span class="synIdentifier">@last_error</span> = e
      erb <span class="synConstant">:error</span>
    <span class="synStatement">end</span>
   <span class="synStatement">end</span>
<span class="synStatement">end</span>
