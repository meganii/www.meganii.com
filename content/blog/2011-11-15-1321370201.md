<p>mechanizeの記事を見かけたので、<a class="keyword" href="http://d.hatena.ne.jp/keyword/Python">Python</a>で<a class="keyword" href="http://d.hatena.ne.jp/keyword/%A5%B9%A5%AF%A5%EC%A5%A4%A5%D4%A5%F3%A5%B0">スクレイピング</a>に挑戦。<br />
<a class="keyword" href="http://d.hatena.ne.jp/keyword/mixi">mixi</a>の過去の日記を<a class="keyword" href="http://d.hatena.ne.jp/keyword/Evernote">Evernote</a>に送りたいなぁと思ってます。</p>

<div class="section">
    <h4>mechanizeとBeautifulSoupのインストール</h4>
    <pre class="code" data-lang="" data-unlink>eary install mechanize
easy install BeautifulSoup</pre>
</div>
<div class="section">
    <h4>過去日記のリスト取得まで</h4>
    <p>とりあえず、<a class="keyword" href="http://d.hatena.ne.jp/keyword/mixi">mixi</a>にログインして、過去日記のリストを取得するまで出来たみたい。</p>
<pre class="code lang-python" data-lang="python" data-unlink><span class="synComment">#!/usr/bin/env python</span>
<span class="synComment">#-*-coding: utf-8 -*-</span>

<span class="synPreProc">import</span> mechanize
<span class="synPreProc">from</span> BeautifulSoup <span class="synPreProc">import</span> BeautifulSoup

MAIL = <span class="synConstant">'hoge@hoge.com'</span>
PASS = <span class="synConstant">'password'</span>

<span class="synComment"># Browser</span>
br = mechanize.Browser()

<span class="synComment"># Browser options</span>
br.set_handle_equiv(<span class="synIdentifier">True</span>)
br.set_handle_redirect(<span class="synIdentifier">True</span>)
br.set_handle_referer(<span class="synIdentifier">True</span>)
br.set_handle_robots(<span class="synIdentifier">False</span>)

<span class="synComment"># User-Agent (this is cheating, ok?)</span>
br.addheaders = [(<span class="synConstant">'User-agent'</span>, <span class="synConstant">'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_7_2) AppleWebKit/535.2 (KHTML, like Gecko) Chrome/15.0.874.120 Safari/535.2'</span>)]

br.<span class="synIdentifier">open</span>(<span class="synConstant">'https://mixi.jp'</span>)
br.select_form(name=<span class="synConstant">'login_form'</span>)

br[<span class="synConstant">'email'</span>] = MAIL
br[<span class="synConstant">'password'</span>] = PASS
response = br.submit()

responsediary = br.<span class="synIdentifier">open</span>(<span class="synConstant">'http://mixi.jp/list_diary.pl?from=navi'</span>)
soup = BeautifulSoup(responsediary.read())
diaryHistory = soup.find(<span class="synConstant">'div'</span>, attrs={<span class="synConstant">'id'</span>:<span class="synConstant">'bodyMainAreaSub'</span>}).find(<span class="synConstant">'div'</span>,attrs={<span class="synConstant">'class'</span>:<span class="synConstant">'diaryHistory'</span>})
historylinks = diaryHistory.findAll(<span class="synConstant">'a'</span>)
<span class="synStatement">for</span> link <span class="synStatement">in</span> historylinks:
    <span class="synIdentifier">print</span> link


</pre>
</div>
