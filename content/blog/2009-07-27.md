<p>とりあえずまずは、<a class="keyword" href="http://d.hatena.ne.jp/keyword/ruby">ruby</a>の環境を整えるところから。思いだしながら書いているので、若干おかしな部分があるかもしれません。</p>
<pre class="code" data-lang="" data-unlink>sudo apt-get ruby</pre><pre class="code" data-lang="" data-unlink>sudo apt-get rubygems</pre><p>このような環境で以下のmain.rbを動かそうとしたところ、なんやら色々エラーを吐いた。</p>
<pre class="code" data-lang="" data-unlink>in `gem_original_require&#39;: no such file to load -- net/https </pre><p>色々調べて見ると、libopenssl-<a class="keyword" href="http://d.hatena.ne.jp/keyword/ruby">ruby</a>というパッケージが足りないらしい。早速、</p>
<pre class="code" data-lang="" data-unlink>sudo apt-get install libopenssl-ruby</pre><p>してあげ、実行すると・・・見事、<a class="keyword" href="http://d.hatena.ne.jp/keyword/twitter">twitter</a>に<a class="keyword" href="http://d.hatena.ne.jp/keyword/API">API</a>を通じて、投稿されました！！<br />

<a class="keyword" href="http://d.hatena.ne.jp/keyword/mixi">mixi</a>の日記を<a class="keyword" href="http://d.hatena.ne.jp/keyword/%C1%B4%CA%B8%B8%A1%BA%F7">全文検索</a>するスプリクトを書いてみた<br />
<a href="http://d.hatena.ne.jp/con_mame/20080726">http://d.hatena.ne.jp/con_mame/20080726</a><br />
