<p>って、あるB4って自分のことなんですけどね・・・昨日の研究室内ミーティングで、発表したときにかなりボロボロで色々とアド<a class="keyword" href="http://d.hatena.ne.jp/keyword/%A5%D0%A5%A4%A5%B9">バイス</a>を頂きました。ありがとうございます。</p><p>そのときのもらったアド<a class="keyword" href="http://d.hatena.ne.jp/keyword/%A5%D0%A5%A4%A5%B9">バイス</a>を自分へのメモということで、形に残していこうと思います。</p>

<div class="section">
    <h4>発表についてのアド<a class="keyword" href="http://d.hatena.ne.jp/keyword/%A5%D0%A5%A4%A5%B9">バイス</a></h4>
    <p><span style="font-size:xx-large;">「相手に伝える気持ちで！！」</span></p>

<ul>
<li>まず、わかりにくい。</li>
<li>自分だけにしか分からない言葉を使っている。</li>
<li>表、グラフを相手に伝えようとしていない。</li>
</ul><p>これは確かに、そうだと思いました。と同時にまず自分が完璧に理解していないのが大きいと感じた。</p>

<ul>
<li>言葉の定義があいまい。</li>
</ul><p>確かに、なんの前置きもなく使ったこと言葉が多く、自分なりの言葉の使い方をしてしまっていた。また本来の言葉の定義と違う意味で使った言葉があったので、誤解を生んでしまっていた。また、厳密な評価式の設定ができていなかった。</p>

<ul>
<li><a class="keyword" href="http://d.hatena.ne.jp/keyword/gnuplot">gnuplot</a>の使い方の工夫が出来る</li>
</ul>
<ol>
<li>得たいグラフを元に<a class="keyword" href="http://d.hatena.ne.jp/keyword/%A5%B9%A5%AF%A5%EA%A5%D7%A5%C8">スクリプト</a>を組む。</li>
<li>グラフの上部に #で<a class="keyword" href="http://d.hatena.ne.jp/keyword/%A5%B3%A5%E1%A5%F3%A5%C8%A5%A2%A5%A6%A5%C8">コメントアウト</a>すれば、<a class="keyword" href="http://d.hatena.ne.jp/keyword/gnuplot">gnuplot</a>からは読み取らない。</li>
<li>そのに実験のメモを残しておいたり、何番目を使えばどんな情報を得たいのかを載せておく。e.g. fitness [0] gene [1]など。</li>
</ol><p>これがビフォー。単純に<a class="keyword" href="http://d.hatena.ne.jp/keyword/gnuplot">gnuplot</a>でplotしただけですね。発表で載せたグラフはこんな感じでした。<br />
軸が何を表しているのか見えないですね・・・<br />
<span itemscope itemtype="http://schema.org/Photograph"><a href="http://f.hatena.ne.jp/meganii/20091209111529" class="hatena-fotolife" itemprop="url"><img src="https://cdn-ak.f.st-hatena.com/images/fotolife/m/meganii/20091209/20091209111529.png" alt="f:id:meganii:20091209111529p:image" title="f:id:meganii:20091209111529p:image" class="hatena-fotolife" itemprop="image"></a></span><br />
これがアフター。表題や、その線が何を表しているのかわかりやすいですね。数字も大きくなっているので、見やすいです。<br />
<span itemscope itemtype="http://schema.org/Photograph"><a href="http://f.hatena.ne.jp/meganii/20091209111527" class="hatena-fotolife" itemprop="url"><img src="https://cdn-ak.f.st-hatena.com/images/fotolife/m/meganii/20091209/20091209111527.png" alt="f:id:meganii:20091209111527p:image" title="f:id:meganii:20091209111527p:image" class="hatena-fotolife" itemprop="image"></a></span></p><p>このグラフにさらに見て欲しい部分を書いておけばバッチリというわけです。</p><p>次の発表のときには、もうちょっとわかりやすい発表を心がけます。</p><br />
<p>また、scriptを使ってplot→グラフ作成をもうちょっと勉強してきます！！</p>
<pre class="code lang-sh" data-lang="sh" data-unlink><span class="synComment">#!/bin/sh</span>

gnuplot <span class="synStatement">&lt;&lt;EOF</span>
<span class="synConstant">set xlabel 'Genelation'</span>
<span class="synConstant">set ylabel 'MaxFitness'</span>

<span class="synConstant">set xlabel font 'Times,30'</span>
<span class="synConstant">set ylabel font 'Times,30'</span>

<span class="synConstant">set xtics font 'Times,20'</span>
<span class="synConstant">set ytics font 'Times,20'</span>

<span class="synConstant">set size 0.6,0.6</span>

<span class="synConstant">set term post enhanced color</span>

<span class="synConstant">set output 'result.eps'</span>

<span class="synConstant">plot '500.dat' u 1:2 w l ti 'MaxFitness'</span>
<span class="synStatement">EOF</span>
</pre>
