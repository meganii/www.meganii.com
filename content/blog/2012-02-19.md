<p><h4>zencodingのダウンロード</h4><a href="https://github.com/rooney/zencoding">rooney/zencoding - GitHub</a></p>
<pre class="code" data-lang="" data-unlink>cd Dropbox/emacs/site-lisp/
git clone https://github.com/rooney/zencoding.git</pre><p><h4><a class="keyword" href="http://d.hatena.ne.jp/keyword/yasnippet">yasnippet</a>のダウンロード</h4><a href="https://github.com/capitaomorte/yasnippet">capitaomorte/yasnippet - GitHub</a><br />
</p>
<pre class="code" data-lang="" data-unlink>cd Dropbox/emacs/site-lisp/
git clone https://github.com/capitaomorte/yasnippet</pre><p><h4><a class="keyword" href="http://d.hatena.ne.jp/keyword/yasnippet">yasnippet</a>の設定(<a class="keyword" href="http://d.hatena.ne.jp/keyword/emacs">emacs</a>.el)</h4></p>
<pre class="code" data-lang="" data-unlink>(add-to-list &#39;load-path &#34;~/Dropbox/emacs/site-lisp/yasnippet&#34;)
(require &#39;yasnippet)
(yas/initialize)
(yas/load-directory &#34;~/Dropbox/emacs/snippets/&#34;) ;snippetsのディレクトリを指定</pre><p><br />
<h4>zencodingの設定(<a class="keyword" href="http://d.hatena.ne.jp/keyword/emacs">emacs</a>.el)</h4><p>zencoding-expand-yas を指定するのがポイントみたい。</p></p>
<pre class="code" data-lang="" data-unlink>(add-to-list &#39;load-path &#34;~/Dropbox/emacs/site-lisp/yasnippet&#34;)
(require &#39;zencoding-mode)
(add-hook &#39;sgml-mode-hook &#39;zencoding-mode) ;; Auto-start on any markup modes
(add-hook &#39;html-mode-hook &#39;zencoding-mode)
(add-hook &#39;text-mode-hook &#39;zencoding-mode)
(define-key zencoding-mode-keymap (kbd &#34;&lt;C-return&gt;&#34;) &#39;zencoding-expand-yas)</pre><p><br />
<h4>Zencodingのカーソル移動がいい感じに！！</h4><a class="keyword" href="http://d.hatena.ne.jp/keyword/Emacs">Emacs</a>を再起動させて、</p>
<pre class="code" data-lang="" data-unlink>ul&gt;*li*3</pre><p>とかやったときに、TABで、いい感じのカーソル位置に移動してくれるようになりました。</p><br />
