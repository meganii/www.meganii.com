<p>自分へのメモ書き</p>
<pre class="code lang-sh" data-lang="sh" data-unlink><span class="synStatement">for</span> FILE <span class="synStatement">in</span> <span class="synSpecial">`\ls | </span><span class="synStatement">grep</span><span class="synSpecial"> ^0.002`</span>
</pre><p>先頭が0.002の<a class="keyword" href="http://d.hatena.ne.jp/keyword/%A5%C7%A5%A3%A5%EC%A5%AF%A5%C8">ディレクト</a>リ(ファイル)の文だけループさせる。<br />
<br />
<br />
</p>
<pre class="code lang-sh" data-lang="sh" data-unlink>tee <span class="synSpecial">-a</span> ファイル
</pre><p>で、-a オプションでファイルに追加で書き込み</p>
<pre class="code lang-sh" data-lang="sh" data-unlink><span class="synComment">#!/bin/sh</span>

<span class="synStatement">for</span> FILE <span class="synStatement">in</span> <span class="synSpecial">`\ls | </span><span class="synStatement">grep</span><span class="synSpecial"> ^0.002`</span>
<span class="synStatement">do</span>
    <span class="synStatement">cd</span> <span class="synPreProc">${FILE}</span>
    sh ~/src/script/sort.sh <span class="synStatement">&gt;</span> output.dat
    sh ~/src/script/ave.sh | tee <span class="synStatement">-a</span> ~/src/script/param.dat
    <span class="synStatement">cd</span> ..
<span class="synStatement">done</span>
</pre>
