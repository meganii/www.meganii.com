<div class="section">
    <h4><a class="keyword" href="http://d.hatena.ne.jp/keyword/%BC%D2%C6%E2SNS">社内SNS</a></h4>
    <p>これは、WWW::Mechanizeを使って、Webの操作をエミュレートさせます。普段のネットサーフィンを一つ一つプログラムに落とし込んでいくことで、その動作を再現できるというものらしいです。</p><p>最初、</p>
<pre class="code lang-ruby" data-lang="ruby" data-unlink>agent.page.link_with(<span class="synConstant">:text</span> =&gt; <span class="synSpecial">&quot;</span><span class="synConstant">日記</span><span class="synSpecial">&quot;</span>).click
</pre><p>の部分を</p>
<pre class="code lang-ruby" data-lang="ruby" data-unlink>agent.page.links.text(<span class="synSpecial">&quot;</span><span class="synConstant">日記</span><span class="synSpecial">&quot;</span>).click
</pre><p>とやっていたんだけど、どうやら古いバージョンの使い方のようで、現バージョンじゃ使えなくなっていた。ここでかなりハマった。</p>
<pre class="code lang-ruby" data-lang="ruby" data-unlink><span class="synPreProc">#!/opt/local/bin/ruby</span>
<span class="synComment">#</span>
<span class="synComment"># %%%{CotEditorXInput=AllText}%%%</span>

<span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">rubygems</span><span class="synSpecial">'</span>
<span class="synPreProc">require</span> <span class="synSpecial">'</span><span class="synConstant">mechanize</span><span class="synSpecial">'</span>
<span class="synIdentifier">$KCODE</span> = <span class="synSpecial">'</span><span class="synConstant">uft8</span><span class="synSpecial">'</span>

<span class="synIdentifier">$USERNAME</span> = <span class="synSpecial">&quot;</span><span class="synConstant">hoge@hoge.com</span><span class="synSpecial">&quot;</span>
<span class="synIdentifier">$PASSWORD</span> = <span class="synSpecial">&quot;</span><span class="synConstant">hogehoge</span><span class="synSpecial">&quot;</span>

<span class="synIdentifier">$SUBJECT</span> = <span class="synIdentifier">STDIN</span>.gets.chomp
<span class="synIdentifier">$BODY</span> = <span class="synIdentifier">STDIN</span>.read

<span class="synPreProc">def</span> <span class="synIdentifier">login</span>
  <span class="synIdentifier">@agent</span> = <span class="synType">WWW</span>::<span class="synType">Mechanize</span>.new
  login_page = <span class="synIdentifier">@agent</span>.get(<span class="synSpecial">&quot;</span><span class="synConstant">http://hoge.hoge.jp/</span><span class="synSpecial">&quot;</span>)   <span class="synComment">#会社のSNSのHP</span>
  login_form = login_page.forms[<span class="synConstant">0</span>]
  login_form.username = <span class="synIdentifier">$USERNAME</span>
  login_form.password = <span class="synIdentifier">$PASSWORD</span>
  <span class="synIdentifier">@agent</span>.submit(login_form)

  edit_page = <span class="synIdentifier">@agent</span>.page.link_with(<span class="synConstant">:text</span> =&gt; <span class="synSpecial">&quot;</span><span class="synConstant">日記</span><span class="synSpecial">&quot;</span>).click
  edit_form = edit_page.forms[<span class="synConstant">0</span>]
  edit_page = <span class="synIdentifier">@agent</span>.submit(edit_form)
<span class="synPreProc">end</span>

<span class="synPreProc">def</span> <span class="synIdentifier">edit</span>
  edit_form = <span class="synIdentifier">@agent</span>.page.forms[<span class="synConstant">0</span>]
  edit_form.subject = <span class="synIdentifier">$SUBJECT</span>        <span class="synComment">#ここが題名</span>
  edit_form.body = <span class="synIdentifier">$BODY</span>           <span class="synComment">#ここが本文</span>
  <span class="synIdentifier">@agent</span>.submit(edit_form)
  make = <span class="synIdentifier">@agent</span>.page.forms[<span class="synConstant">0</span>]
  <span class="synIdentifier">@agent</span>.submit(make)
<span class="synPreProc">end</span>
  
login
edit
</pre><p>ってな感じで、作ってみました。</p><p>この３つのスプリクトをCotEditor上で実行すれば、あーら不思議３つ（ほぼ）同時投稿の完成w</p>
