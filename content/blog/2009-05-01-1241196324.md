<p>必要なパッケージのインストール<pre><br />
sudo apt-get install evince ghostscript cmap-<a class="keyword" href="http://d.hatena.ne.jp/keyword/adobe">adobe</a>-japan1 xpdf-japanese</pre>CMap関係のファイルを検索してくれるように<a class="keyword" href="http://d.hatena.ne.jp/keyword/%A5%B7%A5%F3%A5%DC%A5%EA%A5%C3%A5%AF%A5%EA%A5%F3%A5%AF">シンボリックリンク</a>を作成。<pre><br />
sudo mkdir -p /usr/share/poppler/cidToUnicode<br />
sudo ln --symbolic /usr/share/xpdf/japanese/<a class="keyword" href="http://d.hatena.ne.jp/keyword/Adobe">Adobe</a>-Japan1.cidToUnicode \<br />
 /usr/share/poppler/cidToUnicode/<a class="keyword" href="http://d.hatena.ne.jp/keyword/Adobe">Adobe</a>-Japan1<br />
sudo mkdir -p /usr/share/poppler/cMap<br />
sudo ln --symbolic /usr/share/fonts/cmap/<a class="keyword" href="http://d.hatena.ne.jp/keyword/adobe">adobe</a>-japan1 /usr/share/poppler/cMap/<a class="keyword" href="http://d.hatena.ne.jp/keyword/Adobe">Adobe</a>-Japan1</pre></p><p>ここまでで日本語が通るようになるはずが、<pre><br />
sudo mkdir -p /usr/share/poppler/cidToUnicode<br />
sudo ln --symbolic /usr/share/xpdf/japanese/<a class="keyword" href="http://d.hatena.ne.jp/keyword/Adobe">Adobe</a>-Japan1.cidToUnicode \<br />
 /usr/share/poppler/cidToUnicode/<a class="keyword" href="http://d.hatena.ne.jp/keyword/Adobe">Adobe</a>-japan1</pre>と、<a class="keyword" href="http://d.hatena.ne.jp/keyword/Adobe">Adobe</a>-Japan1を小文字の<a class="keyword" href="http://d.hatena.ne.jp/keyword/Adobe">Adobe</a>-japan1として<a class="keyword" href="http://d.hatena.ne.jp/keyword/%A5%B7%A5%F3%A5%DC%A5%EA%A5%C3%A5%AF%A5%EA%A5%F3%A5%AF">シンボリックリンク</a>を作成してしまったため認識されなかった。</p><br />
<p>その時は、原因がわからなかったため、続けて次の作業も行った。<pre>sudo dpkg-reconfigure cmap-<a class="keyword" href="http://d.hatena.ne.jp/keyword/adobe">adobe</a>-japan1</pre>としてから、「標準」だけではなく「随意」と「追加」にもチェックをいれる。</p><br />
<p><a class="keyword" href="http://d.hatena.ne.jp/keyword/%CA%E4%BD%F5%B4%C1%BB%FA">補助漢字</a>(<a class="keyword" href="http://d.hatena.ne.jp/keyword/JIS%20X%200212">JIS X 0212</a>)用のCMapファイルのインストール。  <pre>sudo mkdir /usr/share/poppler/unicodeMap<br />
sudo ln --symbolic /usr/share/xpdf/japanese/<a class="keyword" href="http://d.hatena.ne.jp/keyword/EUC">EUC</a>-JP.unicodeMap \<br />
 /usr/share/poppler/unicodeMap/<a class="keyword" href="http://d.hatena.ne.jp/keyword/EUC">EUC</a>-JP<br />
sudo ln --symbolic /usr/share/xpdf/japanese/<a class="keyword" href="http://d.hatena.ne.jp/keyword/ISO-2022-JP">ISO-2022-JP</a>.unicodeMap \<br />
 /usr/share/poppler/unicodeMap/<a class="keyword" href="http://d.hatena.ne.jp/keyword/ISO-2022-JP">ISO-2022-JP</a><br />
sudo ln --symbolic /usr/share/xpdf/japanese/Shift-JIS.unicodeMap \<br />
 /usr/share/poppler/unicodeMap/Shift-JIS</pre></p><p>これで無事日本語が通るようになった。<br />
