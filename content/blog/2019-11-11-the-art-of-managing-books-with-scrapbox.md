# 大前提

**「お好きになように！」**

Scrapboxは利用方法を規定するのではなく、使いたいように使うべきです。Scrapboxで読書管理をする場合も、「こうしなければならない」と決めつけるのではなく、自分で工夫しながら試していくのがよいと考えます。

とはいえ、ある程度のルール決めがないと運用するときに迷うので、私は「じぶんルール」を決めて適宜改定していこうと思います。

自分の整理のため、これからScrapboxで読書管理をやろうとしている方の参考になればと思い、この「じぶんルール」を書き出してみました。それではどうぞ！


# ゆるふわ「じぶんルール」

## 0. 「後から変えてもよい」というゆるい心構え

- 気が変わったら「後から変えてもよい」という心構えが大事！

Scrapboxではページタイトルを変えても、よしなにリンクを張り替えてくれます。この「後から変えられる」という安心感があるので、気軽にページを変更することができます。


## 1. 書影重要

{{% img src="https://res.cloudinary.com/meganii/image/upload/v1573368405/mediamarker_jqkwmu.png" w="1440" h="798" alt="Scrapboxで読書管理 - 書影重要" %}}


- 書影があると視覚的に分かるので楽しい
- どんどんページが賑やかになっていくのが楽しい

書影はあった方がテンションが上がります。また、視覚的に必要な情報を探しやすくなるので、機能的にもプラスに働きます。


## 2. 書籍は（）書きで出版社も入れる

{{% img src="https://res.cloudinary.com/meganii/image/upload/v1573650184/mediamarker-naming_tdwyi4.png" w="724" h="425" alt="Scrapboxで読書管理 - ページの名前" %}}

- 小説、アニメ、漫画、映画で名前衝突を防ぐ
- シリーズものはリンクで繋げる

よく小説 => 漫画化 => アニメ化 => 映画化というメディアミックスが展開されます。ページ名を作品名にしてしまうと、１ページにいろんなメディアの情報が混ざってしまいます。そこで、ここでは出版社の情報を付加することで、メディアを一意に識別します。

## 3. 巻数があるものは、ナンバリング(1)のようにナンバリングする

{{% img src="https://res.cloudinary.com/meganii/image/upload/v1573648156/mediamarker-numbering_pqqgdj.png" w="738" h="392" alt="Scrapboxで読書管理 - ナンバリング" %}}

- 無印のページは全巻通したときの概要。ナンバリングしたページは、各巻の感想、読書メモを記載する。

これも名前衝突を防ぐため、複数巻数があるのであれば、ナンバリングします。


## 4. いつでもリファクタリングの心構え

{{% img src="https://res.cloudinary.com/meganii/image/upload/v1573653918/related-links_e9xnyc.png" w="811" h="615" alt="Scrapboxで読書管理 - リファクタリング" %}}

- ある本を読んでいて、別の書籍のことに想いを巡らせた瞬間、その時の気持ちを別の書籍のページに綴る

別にタイトルを変えたくなったら変えても全然問題ありません。前述の通り、Scrapboxがよしなにリンク先を張り替えてくれます。

ある本のメモを取っているときに、別の本を読んだときの自分の考えを知りたくなることもあると思います。そんなときには、脱線して、別の本のページに映り、別の本を読んだときの反論・同意などの情報を追記します。


## 5. 著者の意見と自分の意見を区別する

- 自分の意見の場合は、文の最初、もしくは行末にアイコンを入れる

これは、後で見返したときに、著書の意見なのか、自分の意見を明確にするためです。


## 6. タグをつけるときは、粒度の小さい順にする

関連ページは、タグ・リンクの出現順になるため、初めから #本 など大きい粒度のタグにしてしまうと、大量の関連ページが最初に出てしまい、大変見辛くなります。

よって、粒度の小さいタグから並べるようにしています。

# Scrapboxで読書管理をする効能

## 読書メモが読書中の書籍に閉じない

ある本を読んでいたときに、別の書籍のページも合わせて更新したくなる。合わせて用語も整理したくなる。

## 無限庭いじりを楽しめる

- リンクを更新することで、関連ページが育っていく
- その行為自体が楽しい
- 読書アウトプットと合わせて、自分が知らなかった用語リストが自動的に作られる

## 本だけでなく、聴いたPodcastや音楽、アニメ、映画なんかも一元管理できる

別に読書管理だけに限らず、映画既読、アニメ既読、食べレポ、行った場所の履歴にも利用できる
今はなき、メディアマーカーの思想「メディアを登録できる」

## あれ、この作者の人の本（漫画）なの？という発見がある

思わぬところでリンクが繋がることがある

## 楽しみ方

- 声優情報をまとめておけば、声優のリストができる

## アイデア

- ★マークを付けてオススメ度を付ける

- 一旦非公開プロジェクトで運用している
- なぜか？

- WebClipとして丸ごと保存しているページの扱い
