Scrapboxの`created`を任意の日付フォーマットに変換したい。

```Deno
import { format } from "https://deno.land/std@0.88.0/datetime/mod.ts";

const created = 1619924935
const date = new Date(created * 1000)
console.log(format(date, "yyyyMMddHHmmss"))
```
