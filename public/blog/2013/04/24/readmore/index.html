Octopressに「続きを読む」のプラグイン導入

April 24, 2013

↓こんな感じで「続きを読む」ように、記事を折りたたむプラグインを導入。

Octopressなら、/plagins ディレクトリに 以下の postmore.rbを設定する。

module PostMore
	def postmorefilter(input, url, text)
		if input.include? "

"
			input.split("

").first + "<p class='more'><a href='#{url}'>#{text}</a></p>"
		else
			input
		end
	end
end
