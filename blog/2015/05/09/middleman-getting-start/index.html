Middlemanでの構築メモ。

MiddlemanにBlog機能を持たせる

素の状態のMiddlemanでmiddleman init new_siteをしたときに、デザインがあまりにもアレだったので、心が折られてしまった。 そこで、以下の記事を参考に、最低限のデザインをもったサイトを構築する。 (何もない状態から、ガシガシ作っていけるひとはいいんだが、自分も心が折れました。。。)

Sass - Middleman を使ってブログを作る (心折られずに) - Qiita 

gem install middleman

middleman init my_new_project --template=blog-bootstrap

cd my_new_project
bower install
bundle install

bundle exec middleman server

middleman init
B!
